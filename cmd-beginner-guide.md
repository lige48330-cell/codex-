# CMD 小白教程

这份文件只教一件事：

**让你第一次敢打开命令行，并且知道该输什么。**

如果你以前完全没用过 `cmd`，先别紧张。  
你不需要会编程，只要会照着输入就行。

---

## 1. 先打开 CMD

在 Windows 里，最简单的方法是：

1. 按 `Win + R`
2. 输入 `cmd`
3. 按回车

你会看到一个黑窗口，这就是 CMD。

---

## 2. 你要先知道：命令一行一行输

刚开始不要复制一大串乱七八糟的东西。  
你可以先一行一行试。

比如先输入：

```bat
dir
```

它会列出当前文件夹里的文件。

---

## 3. 进入你的仓库文件夹

假设你的仓库解压到了桌面，你可以这样进：

```bat
cd /d %USERPROFILE%\Desktop\codex-quickstart-package
```

如果你的文件夹不在桌面，就把路径改成你自己的位置。

进入后，你可以先输入：

```bat
dir
```

如果你看到这些文件，说明进对了：

- `README.md`
- `first-run-guide.md`
- `check-codex-env.ps1`
- `start-here.cmd`

---

## 4. 直接开始最简单的流程

如果你在这个文件夹里，直接输入：

```bat
start-here.cmd
```

它会帮你检查环境，然后告诉你下一步。

---

## 5. 常见命令，先记 4 个就够了

### 看当前文件

```bat
dir
```

### 进入文件夹

```bat
cd 文件夹名字
```

### 返回上一级

```bat
cd ..
```

### 退出窗口

```bat
exit
```

---

## 6. 如果你看到“不是内部或外部命令”

这通常说明：

- 你打错了命令
- 你没有进入正确的文件夹
- 你输入的是一个不存在的文件名

先别慌，最常见的做法是：

1. 再输入一次 `dir`
2. 确认文件名真的在这里
3. 再重新运行
